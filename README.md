# Ava's CSS Demo
> [!WARNING]
> TW: this is a React application.
## Following along
Clone the repo and install dependencies:
```bash
git clone https://github.com/avahajr/css-demo.git
cd css-demo
npm install
```
## Development server
```bash
npm run dev     # vite
```
Starts a live development server on [localhost:5173](http://localhost:5173)

### Exposing the dev server to network
```bash
vite --host
```
Will let others follow along to your development server on a network URL.
